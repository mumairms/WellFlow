# 🌸 WellFlow

WellFlow is an AI-powered menstrual health tracking web application built using PHP, MySQL, Bootstrap, and FPDF.

## Features

- User Authentication (Login/Register)
- Smart Period Cycle Tracking
- AI-Based Next Period Prediction
- Reminder Notification System
- Doctor Consultation Booking
- AI Health Chatbot Assistant
- Calendar View
- PDF Health Reports
- Admin Panel
- Dark Mode
- Progressive Web App (PWA)
- Mobile Responsive Design

## Tech Stack

- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: PHP
- Database: MySQL
- Hosting: InfinityFree
- Email System: PHPMailer
- PDF Reports: FPDF
